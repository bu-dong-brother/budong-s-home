如何查看与运行项目（快速指南）
1. 克隆仓库：   
git clone https://github.com/bu-dong-brother/budong-s-home.git
2. 进入对应项目目录，例如：   
cd budong-s-home/电子商务数据
3. 查看项目 README，按其中的 `requirements.txt` 或 `environment.yml` 安装依赖并打开 Jupyter Notebook / Python 脚本或查看 Tableau 工程文件。
