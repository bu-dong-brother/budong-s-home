## 🛠️ 项目详细介绍

### 1. 🇧🇷 巴西电子商务数据分析 (MySQL + Excel + Tableau)
> **目录**: `/巴西电子商务分析`

在这个项目中，我模拟了真实的商业分析流程，处理了包含 10 万+ 订单的 Olist 公共数据集。
- **主要工作**:
  - 使用 **MySQL** 编写复杂查询，进行多表关联（Orders, Customers, Payments），清洗异常数据并聚合关键指标。
  - 使用 **Excel** 构建 **RFM 模型** (Recency, Frequency, Monetary)，计算用户评分并将客户划分为 8 个价值层级。
  - 使用 **Tableau** 制作运营监控仪表板，展示不同 RFM 层的销售贡献与地理分布。
- **技能点**: SQL Joins/Aggregations, RFM Analysis, Business Intelligence.

### 2. 🛒 电子商务数据清洗与可视化 (Python + Tableau + Excel)
> **目录**: `/电子商务数据`

针对一份原始脏数据进行的标准化处理项目。
- **主要工作**:
  - 利用 **Pandas** 进行大规模数据清洗，处理日期格式错误、填补缺失值。
  - 使用 **Tableau** 探索销售额随时间变化的趋势，识别高退货率产品。
  - 使用 **Excel** 构建 **RFM 模型** (Recency, Frequency, Monetary)，计算用户评分并将客户划分为 8 个价值层级。
- **技能点**: Python Data Cleaning, Data Transformation, Visual Storytelling.

### 3. 🏠 北京房价数据分析-链家 (Python EDA)
> **目录**: `/北京房价数据分析-链家`

从数据获取到深度探索的完整 Python 分析项目。
- **主要工作**:
  - 对抓取的北京二手房数据进行特征工程。
  - 使用 **Matplotlib/Seaborn** 进行单变量和多变量分析，探索房屋面积、位置、楼层与价格的相关性。
- **技能点**: Web Scraping (概念), Statistical Analysis, Python Visualization.
